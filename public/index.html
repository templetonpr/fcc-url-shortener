URL Shortener

To submit a new link to be shortened:

Human users:

Use the form here.

Robot users:

  • POST to 
    /new
    with 
    original_url
  • A successful post will return a json string:
    { "original_url": "your original url", "short_url": "your shortened url"}
  • An error will return a json response with 
    {"error": "Error message"}

Disclaimer: This is a demonstration project. Bookmark shortened links at your own peril.
